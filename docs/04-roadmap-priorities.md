## Method-set phasing
The Method Library (see `02`) is comprehensive in ambition but **phased in delivery**. For each analysis we ship a curated **v1 method set** first, then expand the registry over time (more codes, more research methods, more constitutive models). The MVP v1 method set is in `05`.
